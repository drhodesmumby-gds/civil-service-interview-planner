# Core Project Info
- Project: Civil Service Interview Practice Companion
- Purpose: A structured preparation and practice application designed to help candidates prepare timing, notes, and answers for UK Civil Service interviews using the Success Profiles framework and STARR method.
- Key features: AI Interview Plan Generator, STARR Structure Editor, Mock Interview Runner, Follow-Up Question Predictor, Document Import (PDF/Word), GDS/GOV.UK Theme Toggle.
